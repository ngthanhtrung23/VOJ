
\begin{verbatim}



About array A, size is N (1≤N≤1000000000).  First, all of A is 0. I have Q project, one format of:

1 x y: Set A[x]=y.

2 x y: Let print a integer: sum(A[i], i=x..y).

 

\subsubsection{Input}

The first line: N Q (1≤Q≤100000)

Q line next: k x y.

k is 1 or 2, that is type of project. x,y are integer. If k=1, 1≤x,y≤N. If k=2, 1≤x≤N and -1000000≤y≤1000000

 

\subsubsection{Output}

Each project of type 2, print a integer in a line.

 

\subsubsection{Example}
\begin{verbatim}
\textbf{Input:}\end{verbatim}
\begin{verbatim}
\textbf{

5 8

1 2 10

1 3 5

1 4 -2

2 1 3

2 2 5

1 2 0

2 1 32 2 5}\end{verbatim}
\begin{verbatim}
\textbf{Output:}\textbf{15}\end{verbatim}
\begin{verbatim}
\textbf{13}\end{verbatim}
\begin{verbatim}
\textbf{5}\end{verbatim}
\begin{verbatim}
\textbf{3}\end{verbatim}\end{verbatim}
