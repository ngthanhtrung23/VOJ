Chấm điểm
\begin{itemize}
	\item Trong thời gian thi, bài của bạn sẽ được chấm với một test duy nhất là test đề bài.
\end{itemize}
Giới hạn
Trong tất cả các test:
\begin{itemize}
	\item 1 ≤ T ≤ 4
	\item 1 ≤ N ≤ 25
	\item 1 ≤ $x_{i}$ , $y_{i}$ ≤ N
\end{itemize}

Trong 40\% số test đầu tiên, 1 ≤ N ≤ 10, trong 40\% test tiếp theo, 1 ≤ N ≤ 20.
Ví dụ
\begin{verbatim}
\textbf{Input:}
1
4
1 4
2 1
3 2
4 3

\textbf{Output:}
2
\end{verbatim}
