Giới hạn
\begin{itemize}
	\item Trong 20\% test đầu tiên, K = 2, 1 ≤ M, N ≤ 1000
	\item Trong 20\% test tiếp theo, 2 ≤ M, N, K ≤ 200
	\item Trong 30\% test tiếp theo, 2 ≤ M, N, K ≤ 1000
	\item Trong 30\% test còn lại, 2 ≤ K ≤ $10^{9}$ , 2 ≤ M, N ≤ 1000
\end{itemize}
Example
\begin{verbatim}
\textbf{Input:}
3 3 2
0 0 0
1 1 1
1 1 1

\textbf{Output:}
2
0 1 1
0 0 0\end{verbatim}
